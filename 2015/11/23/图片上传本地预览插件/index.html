  • JavaScript

插件介绍

名称

图片上传本地预览插件 v1.1

介绍

基于jQuery扩展,图片上传预览插件
目前兼容浏览器(IE 谷歌 火狐),不支持safar

参数说明

Img : 图片ID;
Width : 预览宽度;
Height : 预览高度;
ImgType : 支持文件类型;
Callback : 选择文件显示图片后回调方法;

使用方法

HTML部分：

把需要进行预览的IMG标签外
套一个DIV
然后给上传控件ID给予uploadPreview事件
