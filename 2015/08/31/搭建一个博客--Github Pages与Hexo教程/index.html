前言

本人是一枚小菜鸟，没有技术基础，也看了不少网上的教程，自己也尝试了无数遍，毕竟有的教程也不是那么详细，大家用的软件版本也不一样，有些地方总是要靠自己去摸索的（说白了就是百度），所以，今天，我的博客终于搭建成功，我决定重新来一遍，同时，把我搭建遇到的问题和过程记录下来，以供大家参考（有的地方可能不是很对，只是我个人的看法而已），我会尽可能详细的记录整个过程，其中有的资料可能是摘录别人教程的，我在最后面会列出，以感谢这些作者。
