PHP之缩略图

缩略图

什么是缩略图?
缩略图: 比原图要小得多的图, 不单只是 size (长和宽)比较小, 图片的大小(所占磁盘空间)比较小.

缩略图的意义?
a)节省存储的磁盘空间(占用磁盘空间), 不成立
b)节省带宽: 大部分的时候,用户可能不需要查看原图
c)提高加载效率: 大图往往需要更多带宽传输,需要的时间比较长.

PHP之验证码

验证码

1.什么是验证码?
验证码: 一个随机字符串,指的其实只是验证码的内容.
验证码是验证码图片的缩写. 是一张在图片中嵌入了随机字符串的技术. 用来区分人和计算机: 人能非常简单的从图片中读取内容,但是计算机很难. 用来防止恶意访问.
Captcha : Completely Automated Public Turing Test to Tell Computers and Humans Apart (全自动区分计算机和人类的图灵测试)

2.验证码的意义?
因为验证码的本质是区分人和计算机. 防止有部分人员利用计算机对网站进行恶意访问( 不断的利用非正常数据对网站进行访问: 导致网站因为无效数据进行各种运算, 致资源浪费; 深层次使得正常用户无法有效获得资源进行登录系统 [ 很慢 ])

PHP 默认是不支持图片处理的,需要利用扩展实现.

Sphinx中文分词

Sphinx 简介

为什么要用 Sphinx

在 mysql 数据库中，对于如下 sql 语句， select * from xxx where like xxx '%xxx'(以%开头的like查询) ，无法使用到任何索引优化，导致如果数据量非常大，查询速度会非常慢。而这种 sql 语句在很多功能中都要用到，如根据歌词查询歌曲，根据剧情查询电影等。如果要加快查询只能使用第三方软件, Sphinx 和 lucence。

mysql 中也提供了全文索引的功能，但是有两个问题：（1）只有 myisam 引擎支持（2）对中文支持不好。不过现在最新的mysql 5.6 版本中的 innodb 1.2 的版本也同样支持全文索引。
